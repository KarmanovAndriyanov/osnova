# osnova
<h1> Сделал окно авторизации, которое работает в полном обьеме, при нажатии на кноку "нет аккаунта", перекидывает на окно регистрации, где мы можем зарегестрировать пользователя, который сохранится и внесется непосредственно в БД, также сделана проверка, чтобы не было пустых полей.</h1>
